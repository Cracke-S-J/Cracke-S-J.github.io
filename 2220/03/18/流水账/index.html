2020-3-20

  4. 工作上 review 了填词，结果还不错；跟 n 师傅吵着把 gui 修完了。“能不能一次稳稳当当地把一件事干好，不要以活多为理由做不好，到头来还是浪费了自己和大家时间。”
  5. 继续看各种文章，打算酝酿一篇分析 CVE 的博客，看起来劳资也是分析过 CVE 的。
  6. 看 70 页操作系统，准备看计算机网络。
  7. 不要因为看东西不动手，动手包括分析漏洞和写代码，不能停止写代码。（明天要看看那个编译器）
